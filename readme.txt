== Description ==
<strong>WP Nightly</strong>

WP Nightly is a plugin that turns the website dark at night. It also gives your website a new look and feel that may be appealing to many users.

WP Nightly works across all browsers perfectly, so no worry which browser your customers are using.

Features:
- Toggle Switch
- OS Aware
- 3 Button Styles
- 3 Color Palettes

== Installation ==

Install the WP Nightly either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.
